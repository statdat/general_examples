# General 
Scripts showing general bioinformatic pipelines and analyses.

## example_AssignTaxonomy
Runs through the multilocus sequence alignment pipeline. GyrB, RpoB, and RpoD is searched against a type strain reference database and the best hit is used to assign taxonomy.

## example_BioPython
Simple script to learn how SeqIO can be used to manipulate headers, descriptions, and the sequences themselves.

## example_GenomeQualityAssessment
Simple example of how genome quality metrics and contamination/completeness scores can be effectively visualized.
